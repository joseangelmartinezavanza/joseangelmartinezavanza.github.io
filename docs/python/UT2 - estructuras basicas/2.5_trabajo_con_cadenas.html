7Ô∏è‚É£ Funciones de cadenas‚Äã

Las cadenas de texto en Python (str) disponen de una gran variedad de m√©todos integrados que permiten transformarlas, analizarlas y darles formato. Estos m√©todos se aplican directamente sobre la cadena y devuelven un nuevo resultado, ya que las cadenas son inmutables (no se modifican en el lugar).

üü© Cambio de formato y may√∫sculas/min√∫sculas‚Äã

üü¶ B√∫squeda y conteo‚Äã

üü® Eliminaci√≥n y reemplazo‚Äã
