UT1 – "Introducción al lenguaje Python"

1️⃣ ¿Qué es Python?​

Python es un lenguaje de programación de alto nivel, de propósito general y ampliamente utilizado. Fue diseñado inicialmente por Guido van Rossum en 1991 y desarrollado por la Python Software Foundation. Se desarrolló principalmente para mejorar la legibilidad del código, y su sintaxis permite a los programadores expresar conceptos en menos líneas de código.

Su sintaxis está pensada para que el código sea lo más parecido posible al lenguaje humano, lo que permite que las ideas se traduzcan en programas de manera clara y en menos líneas de código en comparación con otros lenguajes. Esto lo convierte en una herramienta ideal tanto para aprender a programar como para desarrollar proyectos complejos

Python es uno de los lenguajes más populares de la actualidad según diferentes encuestas como la de StackOverflow o Github.

2️⃣ Historia y evolución de Python​

El desarrollo de Python comenzó a finales de los años 80 en el Centro de Matemática y Computación (CWI) de los Países Bajos. Guido van Rossum buscaba un lenguaje que combinara la potencia de C con la sencillez de Shell, y que sirviera tanto para automatizar tareas como para resolver problemas más complejos.

En 1991 apareció la primera versión pública, la 0.9.0, que ya incluía características muy avanzadas para su época: manejo de excepciones, funciones y módulos. A lo largo de los años, Python evolucionó en paralelo con el crecimiento de Internet y del software libre, lo que impulsó su expansión a nivel global.

  • Python 2.x (2000) introdujo mejoras significativas, pero generó cierta fragmentación en la comunidad debido a la incompatibilidad con versiones anteriores.
  • Python 3.x (2008) marcó un nuevo inicio: corrigió problemas de diseño y añadió mejoras en Unicode, gestión de memoria y librerías modernas. Actualmente es la versión estándar y recomendada.

Gracias a su filosofía de “There’s only one way to do it” (hay una forma preferida de hacer las cosas), Python ha mantenido un crecimiento constante y se ha consolidado como uno de los pilares de la programación actual.

3️⃣ Características principales de Python​

Python posee una serie de características que lo hacen destacar frente a otros lenguajes y que explican su éxito en la industria y en la educación.

🟩 Sencillez y legibilidad​

Una de las mayores fortalezas de Python es que su sintaxis se basa en la claridad y la simplicidad. En lugar de usar llaves {} o palabras reservadas complicadas, emplea indentación para estructurar el código. Esto obliga a escribir programas bien ordenados y fáciles de leer, incluso para personas que nunca han programado antes.

📌 Ejemplo comparativo:

# Python: estructura clara
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

Frente a un lenguaje como C, que utiliza llaves:

if (edad >= 18) {
    printf("Eres mayor de edad");
} else {
    printf("Eres menor de edad");
}

👉 En Python, el código se parece más al lenguaje humano, lo que reduce la curva de aprendizaje.

🟦 Lenguaje interpretado​

Python no necesita un proceso de compilación como C o Java. Su código se ejecuta directamente línea a línea mediante un intérprete. Esto aporta varias ventajas:

  • Se pueden probar fragmentos de código rápidamente en la consola interactiva.
  • Permite detectar errores de forma inmediata.
  • Favorece el desarrollo rápido y la experimentación.

📌 Ejemplo en práctica: En un cuaderno Jupyter Notebook o la consola de Python puedes escribir:

>>> 5 + 3
8
>>> print("Hola mundo")
Hola mundo

Y obtener la respuesta de inmediato, sin necesidad de compilar ni generar ejecutables.

🟨 Multiplataforma​

Python es independiente del sistema operativo. El mismo código puede ejecutarse en Windows, Linux o macOS sin apenas cambios, siempre que tengamos instalado el intérprete. Esto es especialmente útil en entornos educativos y de trabajo remoto: un alumno con Windows y otro con Linux pueden compartir el mismo código sin problemas.

🟧 Tipado dinámico​

En Python no es necesario declarar el tipo de variable al crearla. El intérprete se encarga de asignarlo según el valor. Esto agiliza la programación, aunque requiere disciplina para evitar errores.

📌 Ejemplo:

x = 10        # entero
x = "Hola"    # ahora es una cadena

👉 Los desarrolladores pueden concentrarse en la lógica del programa sin preocuparse al inicio por tipos estrictos. Sin embargo, Python también admite type hints opcionales para proyectos más grandes y colaborativos:

def suma(a: int, b: int) -> int:
    return a + b

🟥 Versatilidad​

Python es un lenguaje multipropósito, lo que significa que se puede utilizar en campos muy distintos:

  • Desarrollo web: frameworks como Django o FastAPI.
  • Ciencia de datos e inteligencia artificial: librerías como Pandas, NumPy o TensorFlow.
  • Automatización: scripts para renombrar archivos, procesar documentos o gestionar servidores.
  • Ciberseguridad: herramientas de análisis de tráfico, pruebas de penetración.
  • Educación: por su sencillez, es el primer lenguaje en muchos programas de enseñanza.

👉 Esto significa que lo que aprendas en este módulo lo podrás aplicar en distintos contextos profesionales.

🟪 Comunidad activa​

Python cuenta con una de las comunidades más grandes y dinámicas del mundo de la programación.

  • Miles de librerías gratuitas en PyPI (Python Package Index).
  • Documentación oficial extensa y en varios idiomas.
  • Foros y comunidades como Stack Overflow, GitHub o Reddit.

📌 Ejemplo práctico: Si necesitas conectarte a una API web, en lugar de programar todo desde cero, puedes instalar una librería existente con:

pip install requests

Y en tu código usarla directamente:

import requests

respuesta = requests.get("https://api.github.com")
print(respuesta.status_code)

👉 Esto ahorra tiempo y esfuerzo, y es posible gracias a la colaboración global de desarrolladores.

4️⃣ Ventajas y Desventajas de Python​

✅ Ventajas❌ Desventajas
📝 Sintaxis clara y legible🐢 Menor rendimiento que lenguajes compilados
💻 Multiplataforma y versátil💾 Mayor consumo de memoria
🌍 Gran comunidad y recursos📱 Poco usado en apps móviles nativas
🔄 Tipado dinámico y flexible⚙️ Necesidad de intérprete para ejecutar
⚡ Alta productividad en desarrollo🔀 Histórica fragmentación de versiones

5️⃣ Usos comunes de Python​

Una de las razones del éxito de Python es su aplicación en múltiples áreas. Entre los usos más frecuentes se encuentran:

  • Ciencia de datos e inteligencia artificial → con librerías como NumPy, Pandas, TensorFlow o PyTorch, Python se ha convertido en el estándar en este sector.
  • Desarrollo web → frameworks como Django, Flask o FastAPI permiten crear páginas y aplicaciones web de forma rápida y eficiente.
  • Automatización de tareas → ideal para escribir scripts que renombren archivos, gestionen datos o extraigan información de páginas web.
  • Ciberseguridad y análisis forense → Python es muy usado para crear herramientas de auditoría, detección de vulnerabilidades o análisis de tráfico de red.
  • Educación y enseñanza → su sencillez lo convierte en el primer lenguaje que aprenden muchos estudiantes en todo el mundo.
  • Desarrollo de videojuegos → librerías como Pygame permiten crear juegos sencillos y prototipos interactivos.
